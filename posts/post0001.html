The Blog Lives!!

(long live the new blog!)

This is the most basic blog I could think of and its still pretty hacky.

Structure:

  • post-list.json contains a list of post file names
  • index.html has a blank 'li' with an id corresponding to the post file name
  • posts.js compiles a url string based on the file name, runs an ajax query, and appends corresponding 'li'

Next steps:
